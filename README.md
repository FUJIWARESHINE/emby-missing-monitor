\# Emby Missing Monitor



A Python script to monitor missing episodes in your Emby library and send notifications.



\## Features

\- Scan your Emby library for missing episodes

\- Send notifications via webhook (Discord, Telegram, etc.)

\- Track progress and log activities



\## Requirements

\- Python 3.7+

\- requests library



\## Installation

1\. Clone this repository

2\. Install dependencies: `pip install -r requirements.txt`

3\. Configure settings in `config.json`



\## Usage

Run the script: `python main.py`
